Welcome to My GitHub Page

This is a simple website hosted with GitHub Pages.

About

You can use GitHub Pages to host static websites directly from your GitHub repository.

Get Started

  1. Create a repository named username.github.io.
  2. Add your HTML files (like this one).
  3. Enable GitHub Pages in the repository settings.
  4. Visit https://username.github.io to see your site!

Learn more at the GitHub Pages documentation.
